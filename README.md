# Fase 2: Modelado
A partir de las observaciones del apartado anterior, se deben definir **las personas primarias y secundarias**. Se puede emplear cualquiera de los dos procesosvistos en clase (*top-down* o *bottom-up*), pero hay que documentar el proceso seguido. En esta fase se valorará el conjunto de personas y su adecuación al proyectopropuesto, así como el proceso seguido para su elaboración

## Contenidos mínimos

Orientativamente, se esperan entre 2-4 personas, aunque este número puede variar mucho de un proyecto a otro. Estas personas serán utilizadas en las siguientes fases.
Es necesario detallar el proceso que se ha seguido y, además de las personas, incluir todos los artefactos intermedios utilizados durante la fase de modelado.

## Artefactos
- Personas
- Materiales generados durante el proceso de definición de las personas
